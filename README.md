# 🛡️ Phishing Website Detection

A web application that detects whether a website URL is legitimate or phishing. The model analyzes various URL-based features and predicts the likelihood of a phishing attack, helping users identify malicious websites before visiting them.

---

## 📌 Features

- Detects phishing and legitimate URLs
- User-friendly interface
- Fast and accurate classification
- Real-time URL analysis
- Displays prediction results instantly

---

## 🚀 Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Flask / Streamlit (whichever you used)
- HTML, CSS (if applicable)

---

The project uses supervised machine learning to classify URLs based on extracted features such as:

- URL Length
- Presence of HTTPS
- Number of Dots
- Number of Special Characters
- Prefix/Suffix Detection
- Domain Information
- Suspicious Keywords
- Other URL-based security features

---

## 📊 How It Works

1. User enters a website URL.
2. The system extracts important URL features.
3. The trained machine learning model analyzes the features.
4. The application predicts whether the URL is:
   - ✅ Legitimate
   - ⚠️ Phishing
